Блоки макета

Логические секции:

1. HEADER (Шапка)
   └── Лого + Навигация

2. HERO
   └── Заголовок, текст, кнопка, логотипы берендов

3. FEATURES (Преимущества)
   └── Заголовок, 3 карточки с иконками

4. HOW IT WORKS (Как это работает)
   └── 2 колонки: текст + изображение

5. PORTFOLIO (Наши работы)
   └── Сетка проектов

6. SERVICES (Услуги)
   └── Заголовок + 9 карточек преимуществ

7. TESTIMONIALS (Отзывы)
   └── Заголовок + карточка отзыва

8. PRICING (Тарифы)
   └── 3 карточки тарифов

9. NEWS & ARTICLES (Статьи)
   └── Сетка из 2 статей

10. FAQ (Вопросы)
    └── Аккордеон/список вопросок

11. FOOTER (Подвал)
    └── Контакты, навигация, соцсети, копирайт
